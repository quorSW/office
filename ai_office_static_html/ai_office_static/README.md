# AI Office Static

Готовая статическая HTML/CSS/JS-папка для GitHub Pages, Vercel или Telegram Mini App.

## Что это
- пиксельный офис
- агенты ходят по офису
- кофейная зона
- задачи проходят по ролям
- панель команды, задач и логов
- без Python и backend

## Как запустить локально
Просто открой `index.html` в браузере.

## Как залить на GitHub Pages
1. Создай новый репозиторий на GitHub.
2. Загрузи в него все файлы из этой папки.
3. Открой `Settings` → `Pages`.
4. Включи GitHub Pages для ветки `main` и папки `/root`.
5. Через минуту сайт будет доступен по ссылке GitHub Pages.

## Как использовать как Telegram Mini App
1. Сначала опубликуй сайт на GitHub Pages или Vercel.
2. Скопируй публичный `https://` URL.
3. В BotFather открой настройки бота.
4. Выбери `Menu Button` или Mini App URL.
5. Вставь ссылку на сайт.

## Ограничения
Это фронтенд-версия офиса.
Она красивая и рабочая визуально, но без реального Python backend и без настоящих LLM-агентов на сервере.
Позже можно подключить API и сделать её полностью живой.
