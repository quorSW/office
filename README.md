# AI Office — Cozy Pixel

Статическая HTML/CSS/JS версия уютного пиксельного AI-офиса.

## Файлы
- `index.html` — основной экран
- `styles.css` — весь визуальный стиль
- `app.js` — простая анимация и поведение работников

## Запуск
Открой `index.html` в браузере или залей файлы в корень репозитория и задеплой на Vercel/GitHub Pages.

## Что это
Это красивый фронтенд-концепт под Telegram Mini App или веб-офис.
