# AI Office v3 Static

Улучшенная статическая версия офиса:
- агенты сидят за рабочими местами;
- ходят только по коридорам;
- не проходят через стены;
- иногда идут за кофе или в лаунж;
- есть очередь задач, команда и лента событий.

## Запуск локально
Просто открой `index.html` в браузере.

## Деплой на Vercel / GitHub Pages
Достаточно положить `index.html`, `styles.css`, `app.js` в корень проекта.
